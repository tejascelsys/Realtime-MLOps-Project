Technical Workflow

1

Source of Truth

Code and Kubernetes manifests live in GitHub. Large datasets and models are tracked by DVC and stored in Azure Blob Storage.

2

Automated CI

GitHub Actions triggers dvc repro to train the model, pushes artifacts to Azure, and auto-commits the new URI back to Git.

3

GitOps Delivery

ArgoCD watches the repo. When the YAML changes, it automatically triggers a rolling update in the Kubernetes cluster.

4

Secure Inference

KServe pods use an init-container to pull the model from Azure using secrets, serving the model via Nginx Ingress.

5

Ops & ML Monitoring

Prometheus scrapes live inference metrics for Grafana dashboards, while Evidently AI detects data and prediction drift.
